
# 요약해조

사용자가 PDF, PPT, MP4, 또는 동영상 링크를 업로드하면 자동으로 **요약과 연습문제**를 생성해주는 Streamlit 기반 웹앱입니다.

## 🚀 기능
- OpenAI API Key 입력
- PDF, PPT, MP4 등 다양한 파일 업로드
- 동영상/문서 링크 텍스트 읽기
- 요약 생성
- 연습문제 자동 생성

## 🛠️ 실행 방법
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## 📁 파일 구성
- `streamlit_app.py` — 메인 앱 파일
- `requirements.txt` — 필요한 패키지 목록
- `README.md` — 설명 문서
